# Homebridge Edison

With this plugin you can control the light bulbs of https://edison-led.co.jp

# Usage example

Partial contents of `~/.homebridge/config.json`:

```json
{
  "accessories": [
    {
      "accessory": "edison",
      "name": "Lamp",
      "ip": "192.168.1.12",
      "id": "117987783663ab4652b7",
      "key": "6d2878334bcade21"
    }
  ]
}
```
